Key insight: The `subtitle:` YAML field renders as `<p class="subtitle lead">` inside `.quarto-title`. There is no secondary subtitle field in trestles. The Director line must be injected as custom HTML in the body, or via a raw HTML block that Quarto will place inside `.about-contents`. Since trestles separates `.about-entity` (photo+title+links) from `.about-contents` (body), we need CSS to position the Director line visually under the subtitle even though it renders in `.about-contents`.

Approach: Place the Director line as the first element in the body content of index.qmd. Use CSS to style it as a subtitle-like element with smaller font and visual separation from the bio paragraph below.

  ```scss
  /* Hero subtitle and Director line — Phase 5 HERO-01/HERO-02 */
  .quarto-about-trestles .subtitle {
    font-size: 1.05rem;
    line-height: 1.4;
    margin-bottom: 0.25rem;
  }

  .director-line {
    font-family: "DM Sans", system-ui, sans-serif;
    font-size: 0.95rem;

    margin-bottom: 1.25rem;
    padding-bottom: 0.75rem;
  ```

  Rationale for each property:
  - `.subtitle` font-size reduced from Bootstrap's default ~1.25rem to 1.05rem — tighter, more information-dense per user direction
  - `.subtitle` margin-bottom reduced to 0.25rem so Director line sits close to subtitle visually
  - `.director-line` uses DM Sans (heading font) at 0.95rem — slightly smaller than subtitle, clearly secondary
  - `.director-line` color #6B5E55 — the muted brown already used for `.pub-summary` and `.cv-entry-meta`, establishing it as secondary text
  - `.director-line` border-bottom creates a thin visual separator between the title block and bio content
  - `.director-line` margin-bottom 1.25rem gives clean breathing room before bio paragraph without being too generous
