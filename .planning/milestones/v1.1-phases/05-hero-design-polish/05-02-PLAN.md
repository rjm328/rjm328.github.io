  <files>custom.scss</files>
  <action>
  In the `/*-- scss:rules --*/` section of custom.scss, make these targeted modifications to EXISTING rules and add new rules. Read the file first to see Plan 01's additions.

  **Modify existing rules** (change values in place, do not duplicate):

  1. `.pub-entry` — tighten spacing:
     ```scss
     .pub-entry {
       margin-bottom: 1.25rem;    /* was 2rem */
       padding-bottom: 1rem;      /* was 1.5rem */
       border-bottom: 1px solid #EDE9E4;
     }
     ```

  2. `.cv-section` — tighten spacing:
     ```scss
     .cv-section {
       margin-top: 1.5rem;        /* was 2.5rem */
       padding-top: 1rem;         /* was 1.5rem */
       border-top: 1px solid #EDE9E4;
     }
     ```

  3. `.cv-entry` — slightly tighten:
     ```scss
     .cv-entry {
       margin-bottom: 0.75rem;    /* was 1.25rem */
     }
     ```

  **Add new rules** (append after the gold accent rules from Task 1):

  ```scss
  /* Tighter typography and spacing — Phase 5 DSGN-04 */
  h1 { font-size: 1.75rem; }
  h3 { font-size: 1.05rem; }

  p { margin-bottom: 0.75rem; }   /* tighter paragraph spacing (Bootstrap default is 1rem) */

  /* Tighter body line-height for information density */
  body {
    line-height: 1.55;            /* Bootstrap cosmo default is ~1.6; tighter but still readable */
  }

  /* Publication summary — tighten */
  .pub-entry .pub-summary {
    font-size: 0.9rem;            /* was 0.95rem — slightly smaller for density */
    margin-top: 0.15rem;          /* was 0.25rem */
    margin-bottom: 0;
  }

  /* Publication buttons — tighter */
  .pub-buttons {
    margin: 0.25rem 0;            /* was 0.5rem 0 */
  }

  /* Quarto about-trestles — reduce generous padding in trestles layout */
  .quarto-about-trestles .about-entity {
    padding-right: 1.5rem;
  }

  .quarto-about-trestles .about-contents {
    padding-left: 0;
  }

  /* Contact page horizontal rules — thinner, less margin */
  hr {
    border-top: 1px solid #EDE9E4;
    margin: 1rem 0;               /* tighter than Bootstrap default ~2rem */
  }
  ```

  Important: The h2 rule from Task 1 already has border-bottom and padding-bottom. MERGE the font-size into that existing h2 block rather than creating a duplicate. The combined h2 rule should be:

  ```scss
  h2 {
    font-size: 1.25rem;
    border-bottom: 2px solid #C4813D;
    padding-bottom: 0.3rem;
    margin-bottom: 0.75rem;
  }
  ```

  Heading hierarchy rationale:
  - h1 at 1.75rem (used for page titles like "Publications", "CV", "Contact" and the hero name)
  - h2 at 1.25rem (used for section headings like "Peer-Reviewed Articles", "Education", "Research Interests")
  - h3 at 1.05rem (used for subsections like "Research Awards", "Teaching Awards")
  - This creates a clear 1.75 > 1.25 > 1.05 step-down that's tight but distinct
  </action>
  <verify>
    <automated>grep 'margin-bottom: 1.25rem' custom.scss | grep -c 'pub-entry' && grep 'margin-top: 1.5rem' custom.scss | grep -c 'cv-section' && grep -c 'font-size: 1.75rem' custom.scss</automated>
  </verify>
  <done>Publication entry spacing reduced to 1.25rem/1rem. CV section spacing reduced to 1.5rem/1rem. Heading scale set to 1.75/1.25/1.05rem. Body line-height at 1.55. Paragraph margin at 0.75rem. All changes are Distill.pub-direction tighter spacing.</done>
