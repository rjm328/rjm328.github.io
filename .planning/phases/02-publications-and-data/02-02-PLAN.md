**APSA citation format rules (locked decisions):**
- Author order: Last, First M., and First M. Last. (first author inverted, rest normal)
- For 3+ authors: Last, First M., First M. Last, and First M. Last.
- Year follows author list (or "Forthcoming")
- Article title in "double quotes"
- Journal name in *italics* (Markdown)
- Volume(Issue): Pages -- no space before parenthetical
- Em dash for page ranges
- Period at end

**Examples:**
- `McGrath, Robert J. 2025. "American Federalism and Health Policy Spillovers: Insights from the Opioid Crisis." *Publius: The Journal of Federalism* 55(3): 477-511.`
- `McGrath, Robert J. Forthcoming. "Shared Pain, Common Purpose..." *Legislative Studies Quarterly*.`
- `Chen, Brian, William Franko, and Robert J. McGrath. 2025. "Occupational Licensing..." *JPAM* 44(4): 1374-1393.`

**Button rules:**
- Only include a button if the YAML field is a non-empty string
- If doi is "", do NOT write the DOI button
- If pdf is "", do NOT write the PDF button
- If data_url is "", do NOT write the Data button
- If ALL three are empty, do NOT write the `<div class="pub-buttons">` at all

**For the book chapter entry:**
- Citation format: Authors. Year. "Chapter Title." In *Book Title*, ed. Editor Name. Publisher.
- No volume/issue/pages

**For working paper entries:**
- Wrap ALL working paper entries inside `::: {.working-papers}` ... `:::`
- Citation format: Authors. "Title." *Working paper.* (or *Work in progress.* or *Book manuscript in progress.*)
- No year, no journal, no volume/issue/pages
- Still include summary for each
