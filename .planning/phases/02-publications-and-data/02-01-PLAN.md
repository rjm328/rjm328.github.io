**Entry counts from the CV (verified in RESEARCH.md):**
- 22 peer-reviewed articles (type: article) -- includes 2 "Forthcoming" entries
- 1 book chapter (type: chapter)
- 3 working papers / works in progress (type: working-paper)

**For the `authors` field:**
- Always quote all author name strings
- First author: "Last, First M." (inverted for APSA)
- Subsequent coauthors: "First M. Last" (not inverted)
- List ALL coauthors in full on every entry (no truncation -- locked decision)

**For the `year` field:**
- Published articles: integer (e.g., 2019)
- Forthcoming articles: string "Forthcoming" (not null, not integer)
- Undated working papers: null

**For the `doi` field:**
- Attempt DOI lookup for each peer-reviewed article. For articles in major journals (LSQ, JPART, JPAM, JOP, Publius, PRQ, etc.) from 2011-2025, DOIs should be findable via CrossRef patterns.
- Standard DOI pattern: `https://doi.org/10.1111/lsq.NNNNN` (Wiley/LSQ), `https://doi.org/10.1093/jopart/XXXNNN` (Oxford/JPART), etc.
- If a DOI cannot be confidently determined, leave as empty string "". Do NOT guess.
- For book chapters and working papers: leave doi as ""

**For the `summary` field:**
- Write a 1-2 sentence plain-language summary for EVERY entry (IDEN-03 requirement)
- Tone: accessible but authoritative -- "This study finds that..." not "Scientists discovered..."
- No jargon: "state legislatures" not "state legislative chambers"
- Draft from paper titles and any abstract/context available
- These are Claude's drafts -- user reviews before publishing

**For the `type` field:**
- Use exactly: `article`, `chapter`, or `working-paper`
- This field drives section grouping on the publications page (Plan 02)

**For the book chapter:**
- Use `book` field instead of `journal` for the book title
- Include `publisher` field: "Palgrave MacMillan"
- Include `editors` field: "David P. Redlawsk"

**Ordering within the YAML file:**
- Group entries by type: articles first, then chapters, then working papers
- Within each type group: reverse chronological (most recent / Forthcoming first)
